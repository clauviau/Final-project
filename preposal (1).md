# Preproposal

## What idea(s) do you have for your final project?
I want to use Pygame for my final project.

I want to create a Escape Room Game. It will have a Game Menu with three buttons in it : Play, Options, Quit. The Play button will immediatly bring the player to the escape room. The Options button will show a list of room options (for this assignement may only be one option), when the player clicks on the room it allows the player to play it. The Quit button only stops the program.

My room concept is a singular room. It contains a painting/frame, library, desk, locked chest and locked door. The goal is to exit through the door. 

By clicking on the various visible aspects of the room, the player will see mini puzzles or such that will allow to solve all the clues. The clues will eventually allow the player to add a key to their inventory. This will allow the player to open the door.

I would like to make it so when you click on an object, the view zooms on it, and would like to add specific sounds for clicks. 

If I have the time/manage to do it, I might want to put a time limit to complete the room.

When the game is won (or lost if time limit) the screen goes black and "You have won" is written on it. 

At the next click, it brings you back to the game menu.
